# 顧客・ターゲット分析
主要ターゲット
書籍・学術系編集者が最優先です。専門書・教科書・学術書の制作現場では、複雑な概念や手順を一枚の図で整理するイラストレーターへの需要が常にあります。景山さんの「情報整理の巧みさ」が最もストレートに伝わる相手であり、DTP制作会社経由でのアクセスも現実的です。
雑誌・新聞の編集者も重要なターゲットです。特集記事のインフォグラフィックや連載コラムの挿絵では、スタイルの一貫性と情報を整理する力が採用理由に直結します。締め切りサイクルが速いため、一度信頼を得ると継続発注につながりやすい層です。
出版社のアートディレクターは意思決定者への直接アクセスルートとして外せません。装丁や誌面設計の段階から関われる可能性があり、単価・規模ともに大きな仕事につながりやすいです。
サブターゲット（中長期）
医療・科学系媒体は線画との相性が特に高く、複雑な構造を正確かつクリーンに描く需要があります。専門性への信頼が積み上がれば単価も上がりやすい分野です。
ビジネス・経済誌は概念やプロセスの図解ニーズが多く、教育系出版は教材や参考書の改訂に伴う繰り返し発注が期待できます。
全ターゲットに共通する「買う理由」
すべての層に共通しているのは、「複雑な情報をシンプルに見せたいが、自分では描けない・時間がない」という課題です。景山さんのモノクロ・線画スタイルは印刷コストとの相性も良く、版面に馴染みやすいという実務的なメリットも訴求できます。スタイルの統一感そのものが、「この人に頼めば仕上がりがブレない」という安心感を生む強みです。
