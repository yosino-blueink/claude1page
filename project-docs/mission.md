#　ミッションステートメント
景山さんのWebページ制作ミッション
このページが存在する理由（Purpose）

複雑な情報を、シンプルな線で解きほぐす。
景山のイラストを必要としている編集者・出版社に、その力を届ける。


ターゲット像
新聞・雑誌・書籍の編集者や出版社のアートディレクターで、「わかりやすいビジュアルで情報を伝えたい」と考えているが、適切なイラストレーターをまだ見つけられていない人。

このページで伝えるべき2つのこと
① 作品を見てもらう（ポートフォリオ）
線画・モノクロという一貫したスタイルで、「情報整理の巧みさ」が伝わる作品を厳選して見せる。スタイルの統一感そのものが信頼を生む。
② 仕事を依頼したいと思わせる（営業）
得意なジャンル・対応できる仕事・連絡先を明確にし、「この人に頼めばうまくいく」という確信を与える。

成功の定義
ページを訪れた編集者が、

「ああ、こういうイラストが欲しかった」と気づき
作品の質と対応範囲を確認して
問い合わせる気持ちになる

この3ステップが自然に起きること。
